## Recent Updates

Stay current with Snacko`s evolution. Check these highlights from our most recent releases.

<Update label="2024-10-15" description="v1.5.0" tags={["feature", "improvement"]}>

## New Features

- Added client sharing for dietitians: Share personalized diet plans directly with clients via secure links.
- Introduced nutrition goal presets: Quick-setup templates for weight loss, muscle gain, and vegan diets.
- Enhanced mobile app with barcode scanner: Scan packaged foods for instant nutrition logging.

## Improvements

- Optimized dashboard load times by `>40%` for faster access to client tracking.
- Improved search in diet plan library with fuzzy matching for ingredients.

## Bug Fixes

- Fixed sync issues between web and mobile for daily intake logs.
- Resolved notification delays for dietitian client updates.

</Update>

<Update label="2024-09-20" description="v1.4.0" tags={["feature", "bugfix"]}>

## New Features

- Dietitian client management dashboard: Bulk import clients from CSV and assign plans in bulk.
- Personal user mode enhancements: Self-track without a dietitian using AI-suggested adjustments.

## Bug Fixes

- Corrected calorie calculation errors for custom recipes.
- Fixed login issues on Safari browsers.

## Improvements

- Better integration with wearables: Sync steps and activity data automatically.

</Update>

<Update label="2024-08-10" description="v1.3.0" tags={["breaking", "feature"]}>

## New Features

- Real-time collaboration: Dietitians and clients co-edit plans with live updates.
- Advanced reporting: Generate PDF exports of progress charts.

## Breaking Changes

- Updated API endpoints: All nutrition tracking now uses `/v2/track` instead of `/track`.

## Bug Fixes

- Patched security vulnerability in shared plan links.

</Update>
