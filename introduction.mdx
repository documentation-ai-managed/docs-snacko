## Overview

Snacko empowers you to build smarter eating habits through personalized nutrition plans and seamless tracking. Whether you work with a dietitian or manage your goals independently, Snacko leverages behavioral science to make healthier choices effortless. You track intake, monitor progress, and achieve sustainable results on `https://dashboard.example.com`.

Dietitians use Snacko to onboard clients, assign tailored diets, generate plans, and oversee adherence in real-time. Individuals access the same powerful tools for self-guided nutrition management.

## Who Uses Snacko

You choose Snacko if you are:

- **An individual** seeking simple, science-backed nutrition tracking without complexity.
- **A dietitian** needing a platform to scale client management, create plans, and deliver results.

Both access `https://dashboard.example.com` for core features, with dietitians unlocking advanced client tools.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com/signup` and create your account. Verify via email.
  </Step>
  <Step title="Set Goals" icon="target">
    Enter your profile details: age, weight, activity level, and dietary preferences.
  </Step>
  <Step title="Build Plan" icon="clipboard-list">
    Generate your first plan or import from a dietitian. Review daily meals and macros.
  </Step>
  <Step title="Track First Meal" icon="utensils">
    Log breakfast using search or camera. See instant nutrient feedback.
  </Step>
</Steps>

For developers integrating Snacko, fetch your first nutrition data:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/nutrition/log', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      userId: 'user123',
      meal: {
        name: 'Oatmeal with berries',
        calories: 350,
        protein: 12
      }
    })
  });
  const data = await response.json();
  console.log(data);
  ```
  ```python
  import requests

  headers = {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
  }
  data = {
      'userId': 'user123',
      'meal': {
          'name': 'Oatmeal with berries',
          'calories': 350,
          'protein': 12
      }
  }
  response = requests.post('https://api.example.com/v1/nutrition/log', json=data, headers=headers)
  print(response.json())
  ```
</CodeGroup>
